# Running

```bash
npm install 
```
```bash
npm start 
```

# Testing
Before check login endpoint you need register user on
```bash
localhost:5000/api/registration
```
with body
```json
{ 
  "email": "", 
  "password": ""
}
```
Then you can test login on 
```bash
localhost:5000/api/login
```
with body
```json
{ 
  "email": "", 
  "password": ""
}
```
